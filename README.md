```bash
  git clone https://github.com/seuusuario/seuprojeto.git
```
2. Acesse a pasta do projeto:
```bash
  cd seuprojeto
```
3. Instale as dependências:
```bash
  npm install  # ou yarn install
```
4. Inicie o projeto:
```bash
  npm start  # ou yarn start
```
